## Application Implements 4 Image Based Object Detection Models:

  - **Single Shot Detector(SSD) Model.**
  
  - **Faster R-CNN(Region-based Convolutional Neural Networks) Model.**
  
  - **YOLO(You Only Look Once) Model.**

  - **CenterNet Model.**
